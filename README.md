# Gestión de Datos de Países

Aplicación en Python para gestionar países con filtros, ordenamientos y estadísticas.

## Funcionalidades
- Alta, modificación y búsqueda de países
- Filtros por continente
- Ordenamientos
- Estadísticas generales
- Persistencia en CSV

## Ejecución
python main.py
